# Évaluation d'impact d'un programme d'immersion anglaise

## Contexte
Projet académique réalisé en équipe de 3 dans le cadre du cours 
ECN-2260 (Économétrie appliquée), Université de Montréal, 2026.

## Objectif
Évaluer l'effet causal d'un programme d'immersion anglaise sur :
- Le profil-type des participants au programme 
- Les résultats scolaires (anglais et mathématiques)
- La fréquentation scolaire

## Données
Quatre bases de données administratives fusionnées :
- Caractéristiques des élèves et parents
- Résultats aux examens (avant/après)
- Présences quotidiennes
- Identifiants des évaluateurs

## Méthodes utilisées
- Statistiques descriptives
- Score de propension et régression logistique
- Méthode de double différence (diff-in-diff)
